# Contact Management System (CLI)

A simple command-line contact management application built using Python.

## Features
- Add contacts
- View contacts
- Update contacts
- Delete contacts
- JSON file storage
- Input validation
- Error handling
- OOP design

## How to Run

1. Navigate to the project folder:
   cd mini_Project

2. Run the program:
   python main.py

## File Structure

- models.py → Contact model
- manager.py → logic & file handling
- main.py → CLI interface
- contacts.json → Auto-generated storage file
